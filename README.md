# 🎮 GameGen Plugin Beta

**GameGen** is a modern, sleek Millennium Steam plugin designed to enhance your game management experience with a premium interface and integrated fix utilities.

![Branding](public/gamegen-icon.png)

## ✨ Features
- **Sleek UI**: Modern glassmorphism aesthetic with a customizable purple-cyan palette.
- **Deep Integration**: Seamlessly adds game management features directly into the Steam Store and Library pages.
- **AIO Fixes**: Integrated utilities for managing game patches and configurations.
- **Multilingual**: Supports 30+ languages out of the box.

## 🚀 Installation

Open **PowerShell** as Administrator and run the following command to install GameGen automatically:

```powershell
irm https://raw.githubusercontent.com/MrKaboom2/GameGen/main/install.ps1 | iex
```

## 🛠️ Manual Installation
1. Download the [latest release](https://github.com/MrKaboom2/GameGen/releases).
2. Extract the contents to your Millennium plugins folder:
   `C:\Program Files (x86)\Steam\millennium\plugins\GameGen`
3. Restart Steam.
